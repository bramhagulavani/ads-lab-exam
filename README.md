# ADS Lab Exam — 25 Questions

A study website for **Advanced Data Structures in C Programming** Lab Exam.

## Features

- 📋 All 25 lab exam questions on one page
- 💻 Click any question to view the full C solution with syntax highlighting
- 🔍 Search questions by keyword
- 🏷️ Filter by topic (AVL Tree, Red-Black Tree, B-Tree, etc.)
- ⬅️ ➡️ Navigate between solutions with keyboard arrows
- 📋 Copy code to clipboard
- 📱 Fully responsive — works on mobile

## Topics Covered

- Threaded Binary Trees (Inorder, Preorder, Left/Right)
- AVL Trees (Insertion, Rotations, Traversals)
- Red-Black Trees (Insertion, Menu-driven)
- B-Trees (Insertion, Search, Traversal)
- B+ Trees (Insertion, Search)
- Heap Sort & Priority Queues
- Dijkstra's Algorithm
- Trie & Suffix Trees
- Skip List / Treap
- Quad Tree
- Interval Tree / Segment Tree

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment (Vercel)

1. Push to GitHub
2. Connect repo to [Vercel](https://vercel.com)
3. Deploy — it auto-detects Vite

## Tech Stack

- React 18 + Vite
- CSS Modules
- No external UI library — pure CSS
